# 今日进化日报

## 先看结论
今天最值得优先看的，还是两类东西：

1. **能直接改善日常工作流的工具型项目**，代表是 **Claude HUD（`jarrodwatts/claude-hud`）**。
2. **会改变助手行为模式的能力层信号**，代表是 **Proactive Agent**。

前者更接近“拿来就能试、很可能马上有体感”的工具升级；后者更像“如果做对了，会改变整个产品行为方式”的能力方向。相比之下，`gsd-build/get-shit-done` 和 `Gog` 仍然值得跟，但暂时还不到今天就要大幅投入注意力的程度。官方项目像 `Claude Code`、`Codex`、`Gemini CLI`` 依然要看，不过更适合作为基线背景，而不是直接挤占今天的 adopt 位。
